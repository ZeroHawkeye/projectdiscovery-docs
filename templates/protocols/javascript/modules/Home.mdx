## 开发

1. 检出此仓库
1. 安装 mintlify，使用 `npm i -g mintlify@latest`
1. 运行 `mintlify dev`

## 部署 

为了构建最终产品，我们有几个额外的步骤：

1. 构建 JS 协议文档

- `npm install -g jsdoc-to-markdown`
- `./bin/jsdocs.sh`

2. 构建 PDCP API 参考文档

- 要么手动下载最新的 `openapi.yaml`，要么运行 `./bin/download-api.sh`
- 运行 `./bin/generate-api.sh` 生成任何新的 API 文件

3. 部署

在这些之后，Mintlify 会自动处理部署。
