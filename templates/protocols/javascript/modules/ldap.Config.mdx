# 接口: Config

[ldap](/templates/protocols/javascript/modules/ldap).Config

Config 是 ldap 客户端的额外配置

**`示例`**

```javascript
const ldap = require('nuclei/ldap');
const cfg = new ldap.Config();
cfg.Timeout = 10;
cfg.ServerName = 'ldap.internal.acme.com';
cfg.Upgrade = true; // 升级到 tls
```
