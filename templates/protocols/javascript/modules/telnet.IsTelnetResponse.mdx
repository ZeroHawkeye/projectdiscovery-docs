# 接口: IsTelnetResponse

[telnet](/templates/protocols/javascript/modules/telnet).IsTelnetResponse

IsTelnetResponse 是 IsTelnet 函数的响应结果。
这是由 IsTelnet 函数返回的。

**`示例`**

```javascript
const telnet = require('nuclei/telnet');
const isTelnet = telnet.IsTelnet('acme.com', 23);
log(toJSON(isTelnet));
```
