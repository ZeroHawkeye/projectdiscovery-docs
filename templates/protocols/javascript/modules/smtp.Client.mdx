# 类: Client

[smtp](/templates/protocols/javascript/modules/smtp).Client

Client 是用于 nuclei 脚本的最小化 SMTP 客户端。

**`示例`**

```javascript
const smtp = require('nuclei/smtp');
const client = new smtp.Client('acme.com', 25);
```
