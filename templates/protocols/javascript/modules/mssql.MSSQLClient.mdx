## 方法

### Connect

▸ **Connect**(`host`, `port`, `username`): `boolean`

Connect 使用给定的凭据连接到 MS SQL 数据库。
如果连接成功，它返回 true。
如果连接失败，它返回 false 和错误。
函数返回后连接将关闭。

#### 参数

| 名称 | 类型 |
| :------ | :------ |
| `host` | `string` |
| `port` | `number` |
| `username` | `string` |

#### 返回

`boolean`

**`示例`**

```javascript
const mssql = require('nuclei/mssql');
const client = new mssql.MSSQLClient;
const connected = client.Connect('acme.com', 1433, 'username', 'password');
```

### IsMssql

▸ **IsMssql**(`host`, `port`): `boolean`

IsMssql 检查给定的主机是否运行 MS SQL 数据库。
如果主机运行 MS SQL 数据库，它返回 true。
如果主机未运行 MS SQL 数据库，它返回 false。

#### 参数

| 名称 | 类型 |
| :------ | :------ |
| `host` | `string` |
| `port` | `number` |

#### 返回

`boolean`

**`示例`**

```javascript
const mssql = require('nuclei/mssql');
const isMssql = mssql.IsMssql('acme.com', 1433);
```
