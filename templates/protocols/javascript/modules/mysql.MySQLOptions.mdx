# 接口: MySQLOptions

[mysql](/templates/protocols/javascript/modules/mysql).MySQLOptions

MySQLOptions 定义了连接到 MySQL 数据库所需的数据源名称 (DSN) 选项，
以及其他选项，如 Timeout 等

**`示例`**

```javascript
const mysql = require('nuclei/mysql');
const options = new mysql.MySQLOptions();
options.Host = 'acme.com';
options.Port = 3306;
```
