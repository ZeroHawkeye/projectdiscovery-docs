<Tabs>
  <Tab title="Go">
    <Note> 在终端中输入以下命令，使用 Go 安装 ProjectDiscovery 的 httpx。 </Note>

    ```bash
    go install -v github.com/projectdiscovery/httpx/cmd/httpx@latest
    ```
  </Tab>
  <Tab title="Brew">
    ```bash
    brew install httpx
    ```

    <Note>支持 **macOS**（或 Linux）</Note>

  </Tab>
  <Tab title="Docker">
    ```bash
    docker pull projectdiscovery/httpx:latest
    ```

    {/* Docker 特定的使用说明可以在[这里](./running#running-with-docker)找到。 */}

  </Tab>
  <Tab title="GitHub">
   <Note> 在终端中输入以下命令，使用 GitHub 安装 ProjectDiscovery 的 httpx。 </Note>
    ```bash
    git clone https://github.com/projectdiscovery/httpx.git; \
    cd httpx/cmd/httpx; \
    go build; \
    mv httpx /usr/local/bin/; \
    httpx -version;
    ```
  </Tab>
  <Tab title="Binary">
    ```bash
    https://github.com/projectdiscovery/httpx/releases
    ```

    <Tip>
    - 为您的操作系统下载最新的二进制文件。
    - 解压缩可直接运行的二进制文件。
    </Tip>

  </Tab>
</Tabs>

## 安装说明
  - httpx 需要安装最新版本的 [**Go**](https://go.dev/doc/install)
  - 将 Go bin 路径添加到系统路径中。在 OSX 或 Linux 中，在终端使用
  
  ```
  echo export PATH=$PATH:$HOME/go/bin >> $HOME/.bashrc
  source $HOME/.bashrc
  ```
  - 在 Windows 中添加 Go bin 路径，[点击此链接获取说明。](https://www.architectryan.com/2018/03/17/add-to-the-path-on-windows-10/)
  - 二进制文件将位于 `$home/go/bin/httpx`
