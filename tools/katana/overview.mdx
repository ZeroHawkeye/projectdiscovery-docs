## 什么是 **Katana?**

Katana 是一个用 Golang 编写的命令行界面 (CLI) 网络爬虫工具，设计为快速、高效并提供简单的输出。
它旨在爬取网站以收集信息和端点。它的一个显著特点是能够使用无头浏览来爬取应用程序。
这意味着它可以爬取使用 JavaScript、Angular 或 React 等技术构建的单页应用程序 (SPAs)，从而有效地访问和收集这些类型应用程序的信息。

 - 查看[这篇精彩的 ProjectDiscovery 博客文章](https://blog.projectdiscovery.io/introducing-katana-the-best-cli-web-crawler/)，了解有关 Katana 初始发布的更多信息。

 - 我们还[撰写了一篇精彩的"深度探索"博客](https://blog.projectdiscovery.io/a-deep-dive-on-katana-field-extraction/)，介绍 Katana 的字段提取功能

## 特性和功能

 - 快速且完全可配置的网络爬取
 - 支持标准模式和无头模式
 - JavaScript 解析和爬取支持
 - 可自定义的自动表单填充
 - 通过预配置字段实现可自定义输出
 - 通过预配置字段和正则表达式实现可自定义范围控制
 - 支持通过 STDIN、URL 和 LIST 输入
 - 支持 STDOUT、FILE 和 JSON 输出
