---

<Tabs>
  <Tab title="Go">
    ```bash
    go install -v github.com/projectdiscovery/nuclei/v3/cmd/nuclei@latest
    ```

    <Note>Nuclei 需要最新版本的 **GO** 才能成功安装。</Note>

  </Tab>
  <Tab title="Brew">
    ```bash
    brew install nuclei
    ```

    <Note>支持 **macOS**（或 Linux）</Note>

  </Tab>
  <Tab title="Docker">
    ```bash
    docker pull projectdiscovery/nuclei:latest
    ```

    Docker 特定的使用说明可以在[这里](./running#running-with-docker)找到。
  </Tab>
  <Tab title="GitHub">
    ```bash
    git clone https://github.com/projectdiscovery/nuclei.git; \
    cd nuclei/cmd/nuclei; \
    go build; \
    mv nuclei /usr/local/bin/; \
    nuclei -version;
    ```

    <Note>Nuclei 需要最新版本的 **GO** 才能成功安装。</Note>

  </Tab>
  <Tab title="Binary">
    ```bash
    https://github.com/projectdiscovery/nuclei/releases
    ```

    <Tip>
    - 下载适用于您操作系统的最新二进制文件。
    - 解压缩即可运行的二进制文件。
    </Tip>

  </Tab>
  <Tab title="Helm">
    ```bash
    git clone https://github.com/projectdiscovery/nuclei.git
    cd nuclei/helm
    helm upgrade --install nuclei . -f values.yaml 
    ```

    <Tip>
    这个 Helm chart 创建了两个主要资源（可以通过 `values.yaml` 进行配置）：

      - 一个 Kubernetes CronJob，按照定义的计划运行 Nuclei

      - 一个供 Nuclei 使用的 [Interactsh](https://github.com/projectdiscovery/interactsh) 服务
    </Tip>

</Tab>
</Tabs>
