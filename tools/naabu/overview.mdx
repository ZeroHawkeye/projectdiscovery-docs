# 什么是 **Naabu?**

Naabu 是一个用 Go 编写的端口扫描工具，能够以快速可靠的方式枚举主机的有效端口。它是一个非常简单的工具，可以对主机或主机列表进行快速的 SYN/CONNECT/UDP 扫描，并提供所有有回应的端口。

## 特性和功能

 - 快速简单的 **SYN/CONNECT/UDP** 探测式扫描
 - 针对易用性进行优化，资源占用**轻量级**
 - **DNS** 端口扫描
 - DNS 端口扫描的**自动 IP 去重**
 - **IPv4/IPv6** 端口扫描（**实验性**）
 - 使用 Shodan [Internetdb](https://internetdb.shodan.io) 进行**被动**端口枚举
 - **主机发现**扫描（**实验性**）
 - 集成 **NMAP** 进行服务发现
 - 多输入支持 - **STDIN/HOST/IP/CIDR/ASN**
 - 多输出格式支持 - **JSON/TXT/STDOUT**
