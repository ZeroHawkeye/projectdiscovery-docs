## 安装说明
  - Cloudlist 需要安装最新版本的 [**Go**](https://go.dev/doc/install)
  - 将 Go bin 路径添加到系统路径中。在 OSX 或 Linux 系统中，在终端使用
  
  ```
  echo export $PATH=$PATH:$HOME/go/bin >> $home/.bashrc
  source $home/.bashrc
  ```
  - 在 Windows 中添加 Go bin 路径，[点击此链接查看说明。](https://www.architectryan.com/2018/03/17/add-to-the-path-on-windows-10/)
  - 二进制文件将位于 `$home/go/bin/cloudlist`
