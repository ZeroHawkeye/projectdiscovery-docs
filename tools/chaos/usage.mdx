## 获取帮助

使用 `chaos -h` 显示所有帮助选项。

## Chaos 选项

| 参数                       | 描述                              | 示例                                                    |
|----------------------------|------------------------------------------|------------------------------------------------------------|
| `-d`                       | 要查找子域名的域名                | `chaos -d uber.com`                                        |
| `-count`                   | 显示指定域名的统计信息             | `chaos -d uber.com -count`                                 |
| `-o`                       | 将输出写入的文件（可选）           | `chaos -d uber.com -o uber.txt`                            |
| `-json`                    | 以 json 格式打印输出              | `chaos -d uber.com -json`                                  |
| `-key`                     | API 的 Chaos 密钥                | `chaos -key API_KEY`                                       |
| `-dL`                      | 包含域名列表的文件（可选）         | `chaos -dL domains.txt`                                    |
| `-silent`                  | 使输出静默                        | `chaos -d uber.com -silent`                                |
| `-version`                 | 打印当前 chaos 客户端版本          | `chaos -version`                                           |
| `-verbose`                 | 显示详细输出                      | `chaos -verbose`                                           |
| `-update`                  | 更新到最新版本                    | `chaos -up`                                                | 
| `-disable-update-check`    | 禁用自动更新检查                  | `chaos -duc`                                               |
