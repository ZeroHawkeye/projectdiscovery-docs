  <Tab title="Go">
    <Note> 在终端中输入以下命令，使用 Go 安装 ProjectDiscovery 的 Chaos 客户端。 </Note>

    ```bash
    go install -v github.com/projectdiscovery/chaos-client/cmd/chaos@latest
    ```
  </Tab>  
</Tabs>

## 安装注意事项
  - Chaos 需要安装最新版本的 [**Go**](https://go.dev/doc/install)
