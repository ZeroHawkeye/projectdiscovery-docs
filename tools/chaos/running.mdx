## 基本用法

```bash
chaos -d uber.com -silent
```

## API 密钥

您可以通过在 [cloud.projectdiscovery.io](https://cloud.projectdiscovery.io?ref=api_key) 注册或登录来获取您的 API 密钥。

## API 密钥环境变量
您还可以在 bash 配置文件中将 API 密钥设置为环境变量。

```bash
export CHAOS_KEY=CHAOS_API_KEY
```
## 注意事项

- API 的速率限制为每分钟每 IP 60 个请求
- Chaos API **仅**支持域名查询。
