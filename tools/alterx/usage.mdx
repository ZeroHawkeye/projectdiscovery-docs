## 获取帮助

使用 `alterx -h` 显示所有帮助选项。

## AlterX 选项

你可以使用以下命令查看可用的标志和选项：

```console
快速且可自定义的子域名词表生成器，使用 DSL。

用法:
  ./alterx [flags]

标志:
输入:
   -l, -list string[]     用于创建排列的子域名（标准输入、逗号分隔、文件）
   -p, -pattern string[]  生成的自定义排列模式输入（逗号分隔、文件）
   -pp, -payload value    以键值格式替换/使用的自定义负载模式输入 (-pp 'word=words.txt')

输出:
   -es, -estimate      估计排列数量而不生成负载
   -o, -output string  写入修改后的子域名列表的输出文件
   -ms, -max-size int  最大导出数据大小 (kb, mb, gb, tb) (默认 mb)
   -v, -verbose        显示详细输出
   -silent             仅显示结果
   -version            显示 alterx 版本

配置:
   -config string  alterx 命令行配置文件 (默认 '$HOME/.config/alterx/config.yaml')
   -en, -enrich    通过从输入中提取单词来丰富词表
   -ac string      alterx 排列配置文件 (默认 '$HOME/.config/alterx/permutation_v0.0.1.yaml')
   -limit int      限制返回结果的数量 (默认 0)

更新:
   -up, -update                 更新 alterx 到最新版本
   -duc, -disable-update-check  禁用自动 alterx 更新检查
```
