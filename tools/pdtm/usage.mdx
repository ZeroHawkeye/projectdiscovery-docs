## 获取帮助

使用 `pdtm -h` 显示所有帮助选项。

## PDTM 选项


```console

Usage:
  ./pdtm [flags]

Flags:
CONFIG:
   -config string            命令行标志配置文件 (默认 "$HOME/.config/pdtm/config.yaml")
   -bp, -binary-path string  自定义下载项目二进制文件的位置 (默认 "$HOME/.pdtm/go/bin")

INSTALL:
   -i, -install string[]  安装单个或多个项目（用逗号分隔）
   -ia, -install-all      安装所有项目
   -ip, -install-path     将路径添加到 PATH 环境变量

UPDATE:
   -u, -update string[]         更新单个或多个项目（用逗号分隔）
   -ua, -update-all             更新所有项目
   -up, -self-update            更新 pdtm 到最新版本
   -duc, -disable-update-check  禁用自动检查 pdtm 更新

REMOVE:
   -r, -remove string[]  移除单个或多个项目（用逗号分隔）
   -ra, -remove-all      移除所有项目
   -rp, -remove-path     从 PATH 环境变量中移除路径

DEBUG:
   -sp, -show-path          显示当前二进制路径然后退出
   -version                 显示项目版本
   -v, -verbose             显示详细输出
   -nc, -no-color           禁用输出内容着色 (ANSI escape codes)
   -disable-changelog, -dc  禁用在输出中显示发布更新日志
