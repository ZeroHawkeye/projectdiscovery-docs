<Tabs>
  <Tab title="Go">
    <Note> 在终端中输入以下命令，使用 Go 安装 ProjectDiscovery 的 Chaos Client。 </Note>

    ```sh
    go install -v github.com/projectdiscovery/pdtm/cmd/pdtm@latest
    ```
  </Tab>
    <Tab title="Binary">
    ```bash
    https://github.com/projectdiscovery/pdtm/releases
    ```
    <Tip>
    - 下载适用于你操作系统的最新二进制文件。
    - 解压可直接运行的二进制文件。
    </Tip>
  </Tab>  
</Tabs>

## 安装说明
  - PDTM 需要最新版本的 [**Go**](https://go.dev/doc/install)
  - 项目通过下载已发布的项目二进制文件进行安装。这意味着项目只能安装在已发布二进制文件的平台上。
  - 路径 $HOME/.pdtm/go/bin 默认添加到 $PATH 变量中
