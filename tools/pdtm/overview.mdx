ProjectDiscovery Tool Manager 或 PDTM 是一种简单的方式来下载并保持您的 ProjectDiscovery 工具井井有条，易于访问、更新等。对于有兴趣使用多个 ProjectDiscovery 工具的用户，我们建议下载 PDTM，而不是单独下载每个工具的二进制文件。

查看[这篇优质博客](https://blog.projectdiscovery.io/getting-started-with-projectdiscovery-in-linux-and-windows/)，了解如何在 Linux 和 Windows 中开始使用 ProjectDiscovery 工具。

您可以在[这里访问 PDTM 的 GitHub 仓库](https://github.com/projectdiscovery/pdtm)。
